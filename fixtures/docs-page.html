Context Bridge

Context Bridge prepares web pages for AI agents by converting noisy pages into compact evidence chunks.

Installation

Install the package with npm install context-bridge.

Run the server with node dist/index.js.

Configuration

Add the server entry to your MCP client config file.

The server exposes read-only tools for search, reading, and context ranking.

Safety

The default tools do not submit forms, click buttons, or modify remote data.
